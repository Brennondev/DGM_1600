GraphQL

GraphQL is a very interesting query language. The implications of what's basically autofill for data that normally needs to be hunted down seems wildly effecient. You can send a GraphQL query to your API and get exactly what you need, nothing more and nothing less. GraphQL queries access not just the properties of one resource but also smoothly follow references between them.
